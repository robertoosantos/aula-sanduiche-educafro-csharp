- Você instalou a versão mais recente de `.net SDK`

## ☕ Usando o Projeto

Para usar projeto, siga estas etapas:

Faça o clone do repositório

```bash
git clone https://github.com/robertoosantos/aula-sanduiche-educafro-csharp.git
```

Execute os testes

```bash
cd Sanduiche.Test/
dotnet watch test
```
